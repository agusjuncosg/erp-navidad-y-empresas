¿Qué cambia en el código actual?

Lo que NO cambia
  • Todo el HTML/CSS
  • Todos los módulos (sales, inventory, etc.)
  • El router (app.js)
  • El sistema de UI (ui.js)
Lo que cambia
  • store.js — reemplaza localStorage por llamadas a Supabase
  • Se agrega un módulo de login (pantalla de autenticación)
  • Se agrega el cliente de Supabase (1 script tag)
